# m3larch.github.io
Repositório oficial da M3L Architecture. Centraliza a apresentação do padrão M3L (Modular in 3 Layers), seus princípios, objetivos e links para os repositórios por tecnologia, como Laravel, Spring Boot e Node.js. Funciona como porta de entrada do ecossistema, com uma visão clara e objetiva da arquitetura.
